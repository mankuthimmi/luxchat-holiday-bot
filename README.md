# Luxchat Holiday Bot

A Python bot for Luxchat (Matrix) that answers questions about Luxembourg public holidays.

## Features
- Check if a specific date is a holiday
- Get list of holidays for a given year
- Responds to user queries in Luxchat rooms

## Installation
1. Install dependencies: `pip install -r requirements.txt`
2. Configure bot credentials in `luxchat_holiday_bot.py`
3. Add your `holidays.json` file (from Feierdeeg)
4. Run: `python luxchat_holiday_bot.py`

## Usage
- Ask: "Is 25-12-2025 a holiday?"
- Ask: "Holidays in 2025"
